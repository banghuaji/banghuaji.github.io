jquery select2使用js赋值

发表于 2017-10-01
但是遇到一个问题 jquery select2插件 取值和原来的select标签取值都一样。但是给jquery select2赋值就出现了问题，用传统的select赋值的方式给jquery select2赋值发现不起作用。 最后查看了jquery select2的api文档，也网上搜了很多资料，但是都给得jquery select2赋值方式是：$(“#id”).val(“1”).trigger(“change”); 我感觉这种赋值方式是有问题，虽然这样赋值也行，但我个人不推荐这样赋值，而是使用jquery select2 api 里提到的赋值方式：$(“#id”).val(“1”).select2(); 这样就可以了。
